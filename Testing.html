Petunjuk Penggunaan

Alat ini dirancang untuk membantu Anda mencatat dan menganalisis data backtest trading Anda. Berikut adalah langkah-langkah untuk menggunakannya:

  1. Pilih Instrumen: Mulai dengan memilih pasangan mata uang atau instrumen lain dari menu "Pilih Pairs" di bagian "Pengaturan Awal".
  2. Isi Data Perdagangan: Masukkan detail setiap perdagangan pada tabel. Pada tampilan HP, setiap data akan tampil seperti kartu untuk memudahkan pengisian.
  3. Tambah Baris Baru: Klik ikon tambah (+) untuk menambahkan data perdagangan baru.
  4. Penyimpanan Otomatis: Tidak perlu khawatir kehilangan data. Semua yang Anda masukkan akan tersimpan secara otomatis di browser Anda. Saat Anda kembali, data akan tetap ada.
  5. Unduh ke Excel: Setelah selesai memasukkan data, klik tombol "Unduh Excel" untuk menyimpan semua catatan Anda dalam format file CSV.
